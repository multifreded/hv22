[HV22.06] privacy isn't given




Blockchain 101


Disclaimer: The following guide is not specific to this challenge, but generally for interacting with smart contracts. It may or may not help you here.


Feeling lost in the blocks? Don't worry, here's a little guide on how to interact with a smart contract:


    

  • Install Metamask from https://metamask.io/
    • 

  • Open Metamask and create an account
    • 

  • After finishing the setup, click the three dots and open up the expanded view. Navigate to "Networks" and click "Add a Network". Enter the following Details (Network name and Currency symbol can be anything):
    • 





    

  • After that, import the private key from the description by clicking on your Account > Import Account.
    • 

  • You can now start interacting with the a blockchain by visiting remix.ethereum.org. You can create/edit Solidity or Typescript files. If you want to interact with a smart contract, create a .sol file and set the contents as the source code of the contract. In the sidebar, change to "Solidity compiler" and compile the smart contract. Then, select "Deploy" in the sidebar and set "Injected provider" or "External Http Provider" as the Environment. You may need to authorize your account to connect to remix. Finally, fill in the text field next to the "at Address" button with the Contract's address and click it. Now you can interact with the contract in Remix:
    • 





    

  • Sometimes, it may help doing things programmatically. For JavaScript and TypeScript, the popular libraries web3.js and ethers.js are available, for a python solution use Web3.py. Be sure to follow the docs of the respective libraries.
